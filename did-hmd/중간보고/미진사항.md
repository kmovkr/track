## 발급자 미진사항
### AS-IS
 - 스키마 발급 UI미지원

 - 신청자 목록 검색기능 미지원
 - 발급된 신원 목록 검색기능 미지원
 - 관리자 authentication  미지원

### TO-BE
 - 스키마 발급 UI 지원

 - 신원 신청목록 검색기능 제공

 - 관리자 authentication 제공

## 검증자 미진사항
### AS-IS
 - 검증 요구사항 작성 UI 미제공
 - 출입 승인 이후 프로세스 미제공
 - 출입 승인에 연결이 선행되어야 한다
 - 관리자 기능 미제공

### TO-BE
 - 검증 요구사항 작성 UI 제공 및 히스토리관리
 - 출입 프로세스 이후, ID부여로 이력관리 제공
 - 출입승인에 연결 없이  connection-less하게 인증 제공
 - 관리자 기능 제공
